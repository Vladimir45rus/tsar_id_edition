🛡️ Tsar-ID Edition: Secure Unified Authentication Hub
The Sovereign Identity Provider for the "Tsarskiy Soft" Ecosystem

🇷🇺 Описание (RU)
Tsar-ID — это фундамент безопасности всей экосистемы «Царь». Модуль обеспечивает единую точку входа (SSO) для всех приложений, гарантируя защиту пользовательских данных на уровне банковских систем.

🚀 Ключевые функции
Единый аккаунт (SSO): Бесшовный переход между «Царь-Подписками», «Царь-Аптечкой» и другими сервисами.

Биометрический щит: Полная поддержка Passkeys, Touch ID и Face ID для входа без паролей.

Интеллектуальный мониторинг: ИИ-анализ попыток входа в реальном времени.

Защита от брутфорса: Динамическая блокировка подозрительных IP-адресов.

🇺🇸 Project Overview (EN)
Tsar-ID is the core identity management layer designed to provide robust, high-security authentication across the entire Tsarskiy Soft digital landscape.

🚀 Features & Capabilities
Universal SSO: One master account to rule all Tsar ecosystem applications.

Advanced Auth: Native support for Biometrics (Face ID/Touch ID) and passwordless Passkeys.

Zero-Trust Architecture: Every login attempt is verified by an AI-driven security layer.

Eco-Sovereignty: Centralized management of user permissions and subscription statuses.
